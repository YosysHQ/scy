Installation guide
==================

How to install ``scy``.

CAD suite(s)
------------

Get Tabby CAD.

Installing from source
----------------------

Currently requires `YosysHQ/sby at krys/scy_dev <https://github.com/yosyshq/sby/tree/krys/scy_dev>`_.

Follow SBY's installation instructions, then run ``make install`` for SCY.
